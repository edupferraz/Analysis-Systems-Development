# Introdução ao HTML (Elementos e Estrutura)

- Semântica
- Elementos de Conteúdo
  - ```<h1>``` a ```<h6>```;
  - ```<p>```
  - ```<hr>``` (vazio)
  - ```<ul>``` + ```<li>``` - Lista não ordenada
  - ```<ol>``` + ```<li>``` - Lista ordenada
  - ```<dl>``` + ```<dt>``` + ```<dd>``` - Lista de conteúdo com valor
 
- Elementos Semânticos
  - ```<header>``` - Grupo navegacional
  - ```<nav>``` - Área para links de navegação
  - ```<section>``` - Seção genérica
  - ```<article>``` - Conteúdo independente e autocontido
  - ```<aside>``` - Uma seção que é tangencialmente relacionado ao conteúdo
  - ```<footer>``` - Define um rodapé para um documento ou seção

- Elementos de Texto
  - ```<em>``` - Trecho enfatizado/destacado
  - ```<strong>``` - Trecho importante
  - ```<code>``` - Insere trecho de código
  - ```<sub>``` - Subscrito
  - ```<sup>``` - Sobrescrito
  - ```<br>``` (vazio) - Quebra de linha
