- Introduction to hardware
  - Hardware - Physical;
  - Software - Cloud;

  - Binary prefixes - Names or symbols preceding the units bits and bytes;

  - IEC Standard - International Electrotechnical Commission;
  - Input devices x Output devices;

  - Computer:
  - Case (Chassis);
  - Motherboard (MoBo):
    - Sockets, AGP Slot, PCI Slot/
    - AT (older) and ATX:
      - AT: Limited interior space; Low air circulation; Shutdown on button press after OS message;
      - ATX: Addresses AT issues;
  - Cooler;
  - Hard Disk (HD);
  - Graphics Card (Video Card);
  - Sound Card;
  - Optional: Stabilizer and Uninterruptible Power Supply (UPS);

Lesson 05 (September 11, 2023)

- Introduction to software:
  - Proprietary software - Copying, redistribution or modification is prohibited; license purchase;
  - Pirate software - Illegal copies; Crime, viruses, lack of support and developer devaluation;
  - Free software - Started in 1983 that founded the GNU, free operating system:
    - GNU - GNU is not Unix;
    - GNU GPL - GNU General Public License;
    - 4 Freedoms users should have:
      - Freedom to run the program, for any purpose;
      - Freedom to study how the program works;
      - Freedom to redistribute copies, including selling;
      - Freedom to modify the program, and release this information;
